# Bug 04 - Legendas do gráfico sobrepostas

## O que é o problema
As legendas do gráfico de categorias na página inicial estão sobrepostas,
tornando impossível ler os valores.

## Como reproduzir
1. Acessar http://localhost:5173
2. Rolar até o gráfico de rosca na página inicial
3. As legendas aparecem umas em cima das outras

## Impacto
Problema de usabilidade — o usuário não consegue visualizar os dados corretamente.
